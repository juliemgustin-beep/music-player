How to get a Spotify API token

  1. Create an app in the Spotify Developer Dashboard .
  2. Copy your Client ID and Client Secret.
  3. Run this command to generate a client credentials token:
curl -X POST "https://accounts.spotify.com/api/token" \
  -H "Content-Type: application/x-www-form-urlencoded" \
  -d "grant_type=client_credentials&client_id=YOUR_CLIENT_ID&client_secret=YOUR_CLIENT_SECRET"

Paste the returned access_token into SPOTIFY_ACCESS_TOKEN in config.js.

How to deploy to GitHub Pages

  1. Push this folder to a GitHub repository.
  2. Open Settings → Pages in GitHub.
  3. Select Deploy from a branch.
  4. Choose the branch and root folder, then save.
  5. Open the published Pages URL after the deploy finishes.
